## :pencil2: Overview

Learn how to implement "WebAssembly" in the Rust programming language.

## :hammer: Requirement

- macOS
- Rust

## :pushpin: Usage

Git clone
```
$ git clone https://github.com/shumatsumoto/wasm-graphic.git
```
Move to directory
```
$ cd wasm-graphic
```
Build
```
$ wasm-pack build --target web
```
start-up http.server
```
$ python3 -m http.server
```
